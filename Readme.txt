TU7FA 
        Augmented Reality Application for Marketing

Installation :
        TU7FA is Android application you can install it on Android phone as (.apk file)

How to use TU7FA?
        1-place the target in specific place.
        2-open the Application.
        3-press the show Button .
        4-choose a category from the 3 categories available in the application for now.
        5-press the category button you want from (Table lamp - ceiling lamp - wall lamp).
        6-inside the category you can choose which accessory you want to show.
        7-press on the accessory you want to show. 
        8-move the camera within the target to fit and scan it.
        9-your selected accessory will be overlaid on the real scene through you device.
